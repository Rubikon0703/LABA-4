# Задание 1

## Задача 1

### Текст задачи
Дерево содержит строки. Добавить к каждой строке заданный символ в конец
Полный текст задачи

### Алгоритм решения
В начале пользователю предлагается ввести строки для построения дерева. Для этого используется рекурсивная функция readStrings, которая запрашивает ввод строки до тех пор, пока не будет введена пустая строка. Каждая непустая строка добавляется в голову накапливаемого списка. Когда ввод завершён, список переворачивается, чтобы сохранить порядок ввода. Введённые строки выводятся на экран.

Далее проверяется, не пуст ли полученный список. Если список пуст, программа выводит сообщение и завершается. Иначе переходим к вводу символа, который будет добавлен в конец каждой строки. Для этого используется рекурсивная функция readChar, которая запрашивает ввод одного символа. Если введена пустая строка, выдаётся сообщение об ошибке и запрос повторяется. Если введено более одного символа, выдаётся предупреждение и используется первый символ. Полученный символ сохраняется.

Затем с помощью функции fromList и свёртки List.fold из списка строк строится бинарное дерево поиска. Функция вставки insert работает по правилам: если строка меньше значения в текущем узле, она направляется в левое поддерево, если больше – в правое; равные строки игнорируются (дерево не содержит дубликатов). Полученное дерево выводится на экран с помощью функции print, которая выполняет симметричный обход  и печатает все элементы в виде списка.

Далее применяется функция addCharToStrings, которая использует определённую для деревьев функцию map. map рекурсивно обходит дерево и применяет к каждому значению переданную функцию – в данном случае это добавление символа в конец строки (конкатенация с string ch). В результате получается новое дерево с изменёнными строками.

Новое дерево выводится на экран.

# Задание 2
## Задача 1

### Текст задачи

Сформировать список из четных элементов дерева.

### Алгоритм решения

В начале пользователю предлагается ввести целые числа для построения дерева. Используется рекурсивная функция readInts, которая запрашивает ввод числа. Ввод проверяется с помощью Int32.TryParse: если введено корректное целое число, оно добавляется в голову списка, иначе выводится сообщение об ошибке и запрос повторяется. Ввод продолжается до тех пор, пока не будет введена пустая строка. После завершения список переворачивается для сохранения порядка ввода. Введённые числа выводятся на экран.

Если список пуст, программа сообщает об этом и завершается. Иначе с помощью функции fromList и свёртки List.fold строится бинарное дерево поиска из чисел. Правила вставки аналогичны: меньшие числа идут в левое поддерево, большие – в правое, равные не добавляются. Дерево выводится на экран функцией print.

Для получения списка чётных чисел используется функция foldTree, реализующая свёртку (fold) для дерева с обходом in-order. В качестве аккумулятора выступает список. Функция свертки проверяет каждый элемент: если он чётный (остаток от деления на 2 равен 0), элемент добавляется в голову аккумулятора, иначе аккумулятор остаётся без изменений. После завершения обхода полученный список реверсируется, чтобы восстановить порядок следования элементов в дереве (так как добавление в голову переворачивает порядок).

Итоговый список чётных элементов выводится на экран.

# Стерхов Юрий БАС-1 Лабораторная №4

# Задание 1

## Задача 1

### Текст задачи
Дерево содержит строки. Добавить к каждой строке заданный символ в конец
Полный текст задачи

### Алгоритм решения
В начале пользователю предлагается ввести параметры для генерации случайных строк. Для этого используется рекурсивная функция readInt, которая запрашивает целое число и проверяет его с помощью Int32.TryParse. При неверном вводе (не число или число, не удовлетворяющее условию) выводится сообщение об ошибке и запрос повторяется. Сначала запрашивается количество строк — оно должно быть положительным. Затем запрашивается минимальная длина строки (положительное число) и максимальная длина строки (должна быть не меньше минимальной).

После получения всех параметров вызывается функция generateRandomString, которая генерирует случайную строку из строчных латинских букв (коды ASCII от 97 до 122). Длина строки выбирается случайно в заданном диапазоне. С помощью List.init создаётся список из count таких строк. Полученный список выводится на экран.

Далее из списка строк строится бинарное дерево поиска. Для этого используется функция вставки insert и свёртка List.fold. Функция insert рекурсивно обходит дерево: если значение меньше значения в узле, оно идёт в левое поддерево; если больше — в правое; равные значения не добавляются (дерево не содержит дубликатов). В результате получается дерево, построенное из сгенерированных строк. Для вывода дерева на экран применяется функция print, которая преобразует дерево в список с помощью симметричного обхода (toList) и выводит его.

Затем программа запрашивает у пользователя символ, который нужно добавить в конец каждой строки. Для ввода символа используется рекурсивная функция readChar. Если введена пустая строка, выдаётся сообщение об ошибке и запрос повторяется; если введено более одного символа, выдаётся предупреждение и берётся первый символ. Полученный символ сохраняется.

Далее применяется функция addCharToStrings, которая использует определённую для деревьев функцию map. Функция map рекурсивно обходит дерево и применяет к каждому значению переданное преобразование — в данном случае это добавление символа в конец строки. В результате формируется новое дерево, где каждая строка исходного дерева дополнена заданным символом.

Новое дерево выводится на экран

# Задание 2
## Задача 1

### Текст задачи

Сформировать список из четных элементов дерева.

### Алгоритм решения

В начале пользователь вводит параметры для генерации случайных целых чисел. Как и в первом задании, используется рекурсивная функция readInt с проверкой через TryParse. Сначала запрашивается количество чисел — оно должно быть положительным. Затем запрашиваются минимальное и максимальное значения чисел (оба могут быть любыми целыми, но максимальное должно быть не меньше минимального). При вводе недопустимых значений выводится сообщение об ошибке и запрос повторяется.

После получения параметров с помощью generateRandomInt (вызов random.Next(minVal, maxVal + 1)) и List.init создаётся список из count случайных чисел в заданном диапазоне. Список выводится на экран.

Из этого списка с помощью функции insert и свёртки List.fold строится бинарное дерево поиска. Вставка выполняется аналогично первому заданию: меньшие числа направляются в левое поддерево, большие — в правое, равные игнорируются. Полученное дерево выводится функцией print (симметричный обход в виде списка).

Далее для получения списка чётных чисел применяется функция foldTree, реализующая свёртку (fold) для дерева. Аккумулятором служит список. Функция свёртки проверяет каждый элемент: если он чётный (остаток от деления на 2 равен 0), элемент добавляется в голову аккумулятора, иначе аккумулятор остаётся без изменений. После завершения обхода список реверсируется с помощью List.rev, чтобы восстановить исходный порядок следования элементов в дереве (поскольку добавление в голову переворачивает порядок).

Итоговый список чётных элементов выводится на экран.
